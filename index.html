Predicting Covid-19 Patterns

For this project, I analyzed Covid-19 time series data that showed the number of new cases and deaths per day globally. The intended goal of the project was to predict the number of new cases and deaths in the future. To achieve this, I used statistical analysis to remove trends from the data. Then I trained a Linear Regression algorithm over the data. Using this method, I obtained a relative error score of ~.03 for predicting new cases and deaths.

Exploit-DB Project

For this project, I scraped exploit entries from the website https://www.exploit-db.com/. After labeling entries with different kinds of exploits, I used a BOWs approach on the dataset, achieving an accuracy score of 95% for predicting the type of exploit. This project was an early experiment that led into my current thesis work with labeling malware threat reports with MITRE ATT&CK labels.
